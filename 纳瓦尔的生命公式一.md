# 纳瓦尔的生命公式一

如果参照系不同，计算方法就会有所不同。这些不是定义，而是成功的算法。

-> 幸福 = 健康 + 财富 + 良好的人际关系

-> 健康 = 锻炼 + 饮食 + 睡眠

-> 锻炼 = 高强度耐力训练 + 体育运动 + 休息

-> 饮食 = 天然食物 + 间歇性禁食 + 植物

-> 睡眠 = 不要闹钟 + 8~9小时 + 昼夜节律

-> 财富 = 收入 + 财富 x (投资回报率)

-> 收入 = 责任 + 杠杆 + 专长

-> 责任 = 个人品牌 + 个人平台 + 承担风险？

-> 杠杆 = 资本 + 人力 + 知识产权

-> 专长 = 无法通过培训获得的知识

-> 投资回报 = “买入并持有” + 估值 + 安全边际
